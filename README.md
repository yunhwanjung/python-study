## 학습 날짜

## 학습 목표

- 변수를 만들고 사용하는 방법 배우기
- 변수 이름 짓는 규칙 익히기

## 주요 개념

### 변수란?
변수는 "이름표가 붙은 상자"와 같습니다.
- 데이터를 저장하는 공간
- 나중에 다시 꺼내서 사용할 수 있음
- 의미 있는 이름을 붙여서 관리

### 변수 만들기 기본 문법
```python
변수이름 = 값
```

### 예시 코드
```python
# 설교 정보 저장하기
sermon_title = "사랑의 계명"
bible_verse = "요한복음 13장 34절"
sermon_date = "2025년 1월 15일"

# 출력하기
print("제목:", sermon_title)
print("본문:", bible_verse)
print("날짜:", sermon_date)
```

## 변수 이름 규칙

✅ **좋은 예시:**
```python
sermon_title = "믿음의 여정"
member_count = 150
church_name = "은혜교회"
```

❌ **나쁜 예시:**
```python
a = "믿음의 여정"      # 의미 불명확
1sermon = "..."        # 숫자로 시작 불가
sermon-title = "..."   # 하이픈 사용 불가
```
